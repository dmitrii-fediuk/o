# ⠿⟨ … ⟩
## Syntax
<code>⠿⟨ <var>P</var> ⟩</code>
## Meaning
Такая запись определяет `ꑌ𝕊` / `ꑌ𝕋` посредством <var>P</var>: формальной спецификации, которой должны соответствовать элементы этого `ꑌ𝕊` / `ꑌ𝕋`. 
## Parameters   
- <var>`P`</var>: is a `ꑌℙ`.
## Example
<code>⠿ ⟨ натуральные числа ⟩</code>   
≡   
<output>множество натуральных чисел</output>
