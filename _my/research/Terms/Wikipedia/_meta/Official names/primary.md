# Valid use of official names
https://en.wikipedia.org/wiki/Wikipedia:Official_names#Valid_use_of_official_names

An **official English name** is a **candidate for what to call an article**, because somebody presumably uses it. 
**It should always be considered as a possibility**, but **should be used only if it is actually the name most commonly used**.
