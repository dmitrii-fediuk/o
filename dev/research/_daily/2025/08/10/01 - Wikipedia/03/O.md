Понятие, которое описывает `Wi`.
Оно обозначено в заголовке `Wi`.
```

### 1.3.

## 2.

```

## 3.
`Gi` : ⠿~
```
Правила типа `G`.
Они содержат `Ga`. 
```

## 4.
`Ri` ≔ (Новая синтаксическая конструкция для `᛭L`, которая упросила бы запись `Gi`)

## 5.
`Ki` ≔ 
```
Некий уникальный одиночный символ в составе `Ri`, который будет обозначать, что `Ri` описывает именно `Wi`
```

## 6. Требования и пожелания к `Ki`
1) Мои текущие кандидаты в `Ki` (обозначу их совокупно `K1s`):
- `៳`

2) 
2.1) `K1s` мне пока нравятся больше всего: больше всего отвечают тому, что мне нужно.
2.2) `K1s` перечисленны именно в том порядке, в каком они мне нравятся: те, что нравятся мне больше — расположены выше других.
2.3) Мне нравится в `K1s` следующее:
- они лаконичны (визуально не громозки)
- они редки

3) Ты в своих вариантах не упоминай `K1s`: мне нужны новые варианты.
Используй `K1s` лишь как пример удачных вариантов: попробуй найти там закономерности (не явно не пиши о них: кроме твоих вариантов ничего вообще не пиши). 

4) `Ki` должен состоять из одного символа.
5) Предлагай только символы из алфавитов.
5.1) Не используй латинский, греческий и русский алфавиты — они не редкие.
5.2) Не предлагай китейские иероглифы: они слишком громоздкие.
5.3) Хорошая идея — использовать символы из древней письменности
5.4) Ещё одна хорошая идея — использовать символы какой-нибудь далёкой от Западного мира письменности
5.5) Ещё одна хорошая идея — использовать символы из писменности какого-нибудь малопопулярного или умирающего языка. 
5.6) Не предлагай мне буквы финикийского алфавита: по непонятной причине, IntelliJ IDEA отображает их как суффиксы, а не как префиксы (при этом Chrome и Notepad++ отображают их правильно)
5.7) Не предлагай мне буквы иврита: они слишком известны в мире.
6) Не используй обычные цифры: ни арабские (`1`, `2`, `3`), ни римские (`I`, `V`).
7) Не используй символы, которые не в состоянии отображать браузер Chrome версии 138 (`C`).
По этой причине, например, не подходит `ᷠ` — он отображается в `C` знаком вопроса.
8) Не предлагай эмодзи. 
9) Не предлагай варианты с маленькой буквой вверху: типа `Lꜞ`.
10) Не предлагай варианты с маленькой буквой строго под `Bi`: типа `L̥`.
11) Обрати внимание, что `៳` (из `K1s`) — это, по сути, перевёрнутый логотип Википедии.
Неплохо было бы найти подобный символ, только не перевёрнутый.
При этом, разумеется, `W` и `w` не годятся (пункт 5.1 выше).
