# ⠿~ ⟨ … ⟩  ⟪ … ⟫
## Syntax
<code> ⠿~ ⟨ <var>D</var> ⟩ ⟪ <var>E</var> ⟫ </code>

## Meaning
Такая запись определяет a `ꑌ⠿` посредством неточного, примерного словесного описания <var>`D`</var>.    
<var>`E`</var> — опциональный блок, перечисляющая некоторые элементы в качестве примеров.
###
<var>`D`</var> может описывать элементы как во множественном числе, так и в единственном.   
Оба этих варианта записи эквивалентны друг другу: `⠿~ ⟨ balls ⟩` ≡ `⠿~ ⟨ ball ⟩`.
## Example
~~~code
⠿~ ⟨ a social media platform where users can express their opinions independently of the platform owner ⟩ ⟪
```examples
- `Reddit`
- GitHub
- Сайты сети StackExchange
- Авторитетные сайты отзывов типа Trustpilot, sikayetvar.com и т.п.
```
⟫
~~~
Здесь the `ꑌ⠿` «social media platforms» определено неточно, примерно.
